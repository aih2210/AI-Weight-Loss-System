## 🚀 如何使用

### 方法1：微信开发者工具（推荐）

**第1步：下载工具**
```
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

**第2步：导入项目**
1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `项目测试/miniprogram-weight-loss` 文件夹
4. AppID选择"测试号"
5. 点击"导入"

**第3步：运行**
1. 点击"编译"
2. 在模拟器中查看
3. 或点击"预览"扫码在手机上查看

### 方法2：真机预览

1. 在微信开发者工具中点击"预览"
2. 用微信扫描二维码
3. 在手机微信中打开小程序
4. 开始使用！
