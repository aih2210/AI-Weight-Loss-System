## 🔄 自动排除的内容

**不会同步的目录：**
- node_modules/ - npm包
- .git/ - Git仓库
- .vscode/ - 编辑器配置
- __pycache__/ - Python缓存
- dist/, build/ - 构建输出

**不会同步的文件：**
- *.log - 日志文件
- *.tmp - 临时文件
- .DS_Store - 系统文件

## 🛡️ 安全提示

### 镜像模式说明
- 使用 `/MIR` 参数（镜像模式）
- 会删除目标目录中源目录没有的文件
- 确保目标目录只用于测试，不要在其中直接开发

### 备份建议
- 首次使用前备份目标目录
- 重要修改前先备份
- 使用 Git 管理源代码

## 💬 常见问题

**Q: 每次都要手动运行吗？**
A: 是的，需要手动运行。这样可以控制同步时机，避免意外覆盖。

**Q: 可以自动同步吗？**
A: 可以使用 Windows 任务计划程序设置定时同步，详见使用说明文档。

**Q: 同步会删除文件吗？**
A: 会删除目标目录中源目录没有的文件（镜像模式）。

**Q: 如何只同步小程序？**
A: 使用 "快速同步小程序.bat"。

**Q: 同步失败怎么办？**
A: 查看错误提示，参考使用说明文档的故障排查部分。
