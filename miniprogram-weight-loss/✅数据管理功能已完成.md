## 📱 使用方法

### 导出数据
1. 点击"导出数据"
2. 查看数据统计信息
3. 点击"复制数据"
4. 打开备忘录或文本编辑器
5. 粘贴并保存为文件

### 导入数据
1. 复制JSON数据到剪贴板
2. 点击"导入数据"
3. 确认导入信息
4. 等待导入完成
5. 重启小程序

### 本地备份
1. 点击"立即备份"
2. 等待备份完成
3. 需要时点击"恢复备份"
4. 确认恢复操作

### 清除数据
1. 点击"清除所有数据"
2. 第一次确认
3. 第二次确认（最后确认）
4. 等待清除完成
5. 自动重启到初始状态
