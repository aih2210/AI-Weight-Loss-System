## 📊 数据结构

### userData 扩展字段
```javascript
{
  healthCoins: 0,              // 健康币余额
  coinHistory: [],             // 交易历史
  checkInHistory: [],          // 签到历史
  orders: []                   // 订单列表
}
```

### coinHistory 记录
```javascript
{
  type: 'earn' | 'spend',      // 类型
  amount: 10,                  // 金额
  reason: '每日签到',          // 原因
  balance: 100,                // 余额
  timestamp: '2024-01-01'      // 时间
}
```

### checkInHistory 记录
```javascript
{
  date: '2024-01-01',          // 日期
  coins: 10,                   // 获得健康币
  consecutiveDays: 1           // 连续天数
}
```

### orders 记录
```javascript
{
  id: 1234567890,              // 订单号
  product: {...},              // 商品信息
  cashPaid: 199,               // 现金支付
  coinPaid: 500,               // 健康币支付
  status: 'pending',           // 状态
  orderTime: '2024-01-01'      // 下单时间
}
```
