# ⚡ 通义千问快速配置指南

## 🚀 3分钟快速开始

### 步骤1：获取API Key（2分钟）

1. 访问：https://dashscope.aliyun.com
2. 登录/注册阿里云账号
3. 点击"开通DashScope"
4. 进入"API-KEY管理"
5. 点击"创建新的API-KEY"
6. 复制生成的Key（格式：sk-xxxxxx）

💡 **新用户福利**：免费100万tokens额度！

### 步骤2：配置小程序域名（1分钟）

**开发阶段（推荐）：**
1. 打开微信开发者工具
2. 点击右上角"详情"
3. 勾选"不校验合法域名、web-view..."

**正式发布：**
1. 登录微信小程序后台
2. 开发 → 开发管理 → 开发设置
3. 服务器域名 → request合法域名
4. 添加：`https://dashscope.aliyuncs.com`

### 步骤3：在小程序中设置（30秒）

1. 打开AI助手页面
2. 点击右上角 ⚙️ 按钮
3. 粘贴你的API Key
4. 点击确定

✅ **完成！现在可以使用AI对话了！**

## 🔧 常见问题

### Q1: API调用失败？
**检查清单：**
- [ ] API Key是否正确（sk-开头）
- [ ] 网络是否正常
- [ ] 是否勾选"不校验合法域名"（开发阶段）
- [ ] 阿里云账户是否欠费

### Q2: 回答速度慢？
- 正常响应时间：2-5秒
- 如果超过10秒，检查网络
- 可以切换到本地模式（秒回）

### Q3: 想用更强大的模型？
修改代码中的模型参数：
```javascript
// ai-assistant.js 第115行
model: 'qwen-plus',  // 改为 qwen-plus 或 qwen-max
```

### Q4: API Key在哪里存储？
- 存储在微信小程序本地存储
- 只在你的设备上
- 不会上传到服务器
- 可以随时删除或更换

## 📱 使用技巧

### 1. 提问技巧
❌ 不好："怎么减肥"
✅ 更好："我是女生，25岁，想在3个月内减重8kg，请给我详细的饮食和运动计划"

### 2. 充分利用个性化
AI已经知道你的：
- 当前体重
- 目标体重
- 减重进度

直接问："根据我的情况，今天应该吃什么？"

### 3. 连续对话
AI会记住对话上下文：
- 你："推荐晚餐"
- AI："建议吃鸡胸肉沙拉..."
- 你："如果我不喜欢鸡胸肉呢？"
- AI："那可以用鱼肉或豆腐替代..."

### 4. 模式切换
- 简单问题 → 本地模式（快）
- 复杂问题 → AI模式（智能）
- 没网络 → 本地模式（离线）
