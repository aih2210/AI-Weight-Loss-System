## 🎯 新增功能

### 1. **详细食谱数据库**

每个食谱现在包含：

#### 📋 基础信息
- 菜品名称
- 营养数据（卡路里、蛋白质、碳水、脂肪）
- 烹饪时间
- 难度等级
- 份量

#### 🛒 食材清单
每个食材包含：
- 食材名称
- 用量
- 备注说明

示例：
```javascript
{
  name: '鸡胸肉',
  amount: '200g',
  note: '去皮'
}
```

#### 👨‍🍳 详细步骤
每个步骤包含：
- 步骤编号
- 步骤标题
- 详细内容
- 所需时间
- 烹饪技巧

示例：
```javascript
{
  step: 1,
  title: '腌制鸡胸肉',
  content: '鸡胸肉洗净擦干，用刀背轻轻拍打使其松软...',
  time: '10分钟',
  tips: '拍打可以让鸡肉更嫩，腌制时间不要太长'
}
```

#### 💡 烹饪技巧
- 食材选购建议
- 烹饪注意事项
- 替代方案

#### 🌟 营养亮点
- 营养价值说明
- 健康益处
- 适用人群

## 📝 使用流程

### 1. 生成食谱
```
AI食谱生成页面
→ 选择餐次类型
→ 输入目标卡路里（可选）
→ 点击"生成AI食谱"
→ 查看推荐食谱列表
```

### 2. 查看详情
```
点击食谱卡片
→ 进入详情页面
→ 查看营养信息
→ 切换到"食材清单"
→ 准备所需食材
```

### 3. 开始烹饪
```
点击"开始烹饪"
→ 切换到"制作步骤"
→ 按步骤操作
→ 点击步骤标记完成
→ 查看进度条
→ 完成所有步骤
```

### 4. 查看技巧
```
切换到"小贴士"
→ 查看烹饪技巧
→ 了解营养亮点
→ 学习替代方案
```
