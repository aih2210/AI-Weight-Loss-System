# 📱 AI减重App启动指南

## 快速开始

### 方式1：使用Expo Go（最简单，推荐）

1. **安装Expo Go**
   - Android: 在Google Play搜索"Expo Go"
   - iOS: 在App Store搜索"Expo Go"

2. **启动开发服务器**
   ```bash
   cd E:\项目测试\mobile-app
   npm install
   npm start
   ```

3. **扫码运行**
   - 用手机扫描终端显示的二维码
   - 应用会自动在Expo Go中打开

### 方式2：在模拟器中运行

**Android模拟器：**
```bash
npm run android
```

**iOS模拟器（仅Mac）：**
```bash
npm run ios
```

### 方式3：在浏览器中预览
```bash
npm run web
```

确保后端服务正在运行：

- 如需修改API地址，编辑 `src/services/api.ts`

## 常见问题

**Q: npm命令不可用？**
A: 需要先安装Node.js: https://nodejs.org/

**Q: 如何在真机上测试？**
A: 使用Expo Go App扫码即可，无需安装Android Studio或Xcode

**Q: 如何打包成APK/IPA？**
A: 使用 `expo build:android` 或 `expo build:ios`
