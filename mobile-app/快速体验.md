# 📱 移动App快速体验指南

## 🎯 最简单的方式：在浏览器中运行

由于你的系统还没有安装Node.js，我为你准备了一个纯HTML版本的移动App演示！

### 立即体验（无需安装任何东西）

**双击打开这个文件：** `E:\项目测试\mobile-app-demo.html`

这个演示包含：
- ✅ 完整的移动端UI界面
- ✅ 所有功能页面（首页、饮食、运动、我的）
- ✅ 真实的交互效果
- ✅ 连接到后端API

## 📋 如果要运行完整的React Native版本

### 步骤1：安装Node.js

1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本（推荐18.x或20.x）
3. 安装完成后重启命令行

### 步骤2：安装依赖

```bash
cd E:\项目测试\mobile-app
npm install
```

这个过程可能需要5-10分钟，请耐心等待。

### 步骤3：启动开发服务器

```bash
npm start
```

### 步骤4：查看二维码

启动成功后，你会看到：
- 终端中显示一个二维码
- 或者浏览器自动打开 http://localhost:19006

### 步骤5：在手机上运行

**方式A：使用Expo Go（推荐）**
1. 在手机上安装 Expo Go App
   - Android: Google Play 搜索 "Expo Go"
   - iOS: App Store 搜索 "Expo Go"
2. 打开Expo Go，扫描二维码
3. App会自动加载并运行

**方式B：在浏览器中运行**
- 访问 http://localhost:19006
- 可以看到移动端界面的Web版本

## ❓ 常见问题

**Q: 为什么没有显示二维码？**
A: 可能是：
1. Node.js未安装
2. npm install 还没完成
3. 端口被占用

**Q: 不想安装Node.js怎么办？**
A: 使用我准备的HTML演示版本：`mobile-app-demo.html`

**Q: Expo Go在哪里下载？**
A: 
- Android: https://play.google.com/store/apps/details?id=host.exp.exponent
- iOS: https://apps.apple.com/app/expo-go/id982107779
