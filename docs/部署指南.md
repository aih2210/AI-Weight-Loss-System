# 部署指南

## 开发环境部署

### 1. 后端部署

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### 2. 前端部署

```bash
cd frontend
npm install
npm run dev
```

## 生产环境部署

### 使用Docker

```bash
docker-compose up -d
```

### 数据库迁移

```bash
cd backend
alembic upgrade head
```

## 环境变量配置

参考 `.env.example` 文件配置以下环境变量：

- DATABASE_URL
- REDIS_URL
- OPENAI_API_KEY
- SECRET_KEY
