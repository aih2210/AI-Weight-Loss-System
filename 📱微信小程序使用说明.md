## 🚀 如何使用

### 第1步：安装微信开发者工具

1. 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载对应系统版本
3. 安装到电脑

### 第2步：导入项目

1. 打开微信开发者工具
2. 点击"+"或"导入项目"
3. 项目目录选择：`项目测试/miniprogram-weight-loss`
4. AppID选择"测试号"（或填入你的AppID）
5. 项目名称：AI减重助手
6. 点击"导入"

### 第3步：运行项目

1. 导入成功后自动打开项目
2. 点击顶部"编译"按钮
3. 在左侧模拟器中查看效果
4. 可以点击"预览"扫码在手机上查看

## 💾 数据存储

### 存储方式
使用微信小程序的 `wx.setStorageSync` API

### 存储内容
```javascript
{
  user: {
    name: '用户',
    age: 30,
    gender: 'male',
    height: 170,
    currentWeight: 56,
    targetWeight: 50,
    startWeight: 60,
    startDate: '2024-12-26'
  },
  weightHistory: [],    // 体重记录
  foodLogs: [],        // 饮食记录
  exerciseLogs: [],    // 运动记录
  emotionLogs: [],     // 情绪记录
  recipes: [],         // 收藏的食谱
  settings: {}         // 设置
}
```

### 数据操作
```javascript
const app = getApp();

// 获取所有数据
const data = app.getData();

// 添加体重记录
app.addWeightRecord(56, '2024-12-26');

// 添加饮食记录
app.addFoodLog({
  name: '鸡胸肉',
  calories: 248,
  portion: 150,
  meal: '午餐'
});

// 获取今日卡路里
const calories = app.getTodayCalories();

// 获取连续打卡天数
const streak = app.getStreak();
```

## ❓ 常见问题

### Q: 需要AppID吗？
A: 开发测试可以使用"测试号"，发布上线需要注册小程序获取AppID。

### Q: 如何在手机上预览？
A: 在微信开发者工具中点击"预览"，用微信扫描二维码即可。

### Q: 数据会丢失吗？
A: 数据存储在微信本地存储中，除非用户清除微信缓存，否则不会丢失。

### Q: 可以添加后端服务吗？
A: 可以！配置服务器域名后，使用 `wx.request` 调用后端API。

### Q: 如何发布小程序？
A: 需要注册小程序账号，上传代码，提交审核，审核通过后发布。

### Q: 个人可以注册吗？
A: 可以！个人和企业都可以注册小程序。

### Q: 需要服务器吗？
A: 不需要！当前版本所有数据存储在本地，完全离线可用。
