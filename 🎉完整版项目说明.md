## 🎯 核心AI功能

### 1. 个性化代谢建模引擎

**文件：** `backend/models/metabolism_model.py`

**功能：**
- 基础代谢率(BMR)预测
- Harris-Benedict公式计算
- 梯度提升树模型
- 平台期自动检测
- 未来体重变化预测

**API接口：**
```python
POST /api/metabolism/predict-bmr
POST /api/metabolism/detect-plateau
POST /api/metabolism/predict-weight
```

### 2. 情境感知饮食AI顾问

**文件：** `backend/models/diet_ai.py`

**功能：**
- 食物图像识别（模拟）
- 内置10+常见食物数据库
- 营养成分查询
- AI食谱生成（GPT-4）
- 情绪性进食分析
- 社交场景餐厅建议

**内置食物库：**
- 鸡胸肉、糙米饭、西兰花
- 鸡蛋、燕麦、牛奶
- 苹果、香蕉、三文鱼、豆腐

**API接口：**
```python
POST /api/diet/recognize-food
POST /api/diet/log-food
POST /api/diet/log-emotion
GET /api/diet/analyze-emotion-pattern/{user_id}
POST /api/diet/generate-recipe
```

### 3. 自适应运动AI教练

**文件：** `backend/models/exercise_coach.py`

**功能：**
- 个性化运动计划生成
- 基于体能水平调整
- 运动依从性分析
- 最小可行方案（5分钟运动）
- 根据反馈动态调整

**运动类型：**
- 有氧：快走、慢跑、跑步、游泳、骑行
- 力量：深蹲、俯卧撑、平板支撑、哑铃
- 柔韧：瑜伽、拉伸

**API接口：**
```python
POST /api/exercise/generate-plan
POST /api/exercise/log-exercise
GET /api/exercise/analyze-adherence/{user_id}
POST /api/exercise/minimal-plan/{user_id}
```

### 4. 智能推荐引擎

**文件：** `backend/services/recommendation_engine.py`

**功能：**
- 每日个性化建议生成
- 膳食计划自动生成
- 进度分析报告
- 成就系统

**推荐类型：**
- 体重进度分析
- 饮食提醒
- 运动建议
- 平台期检测
- 健康小贴士

**API接口：**
```python
POST /api/recommendations/daily
GET /api/recommendations/meal-plan/{calorie_target}
POST /api/recommendations/progress-report
```
