### 方法2：使用Git命令行

```bash
# 1. 初始化Git
git init

# 2. 配置用户信息（首次使用）
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit: AI智能减重系统完整项目"

# 5. 关联远程仓库（替换为你的仓库URL）
git remote add origin https://github.com/your-username/AI-Weight-Loss-System.git

# 6. 推送
git branch -M main
git push -u origin main
```

## 🐛 遇到问题？

### 常见问题

**Q1: 推送失败，提示"Permission denied"**
- 检查Personal Access Token是否正确
- 确认Token权限包含`repo`

**Q2: 推送很慢**
- 检查网络连接
- 考虑使用代理

**Q3: 提示"Large files detected"**
- 检查是否有大文件未被.gitignore排除
- 使用Git LFS处理大文件

**Q4: 不小心上传了敏感信息**
- 立即删除仓库或设为私有
- 更换所有泄露的密钥
- 清理Git历史

详细解决方案请查看：[📤GitHub上传指南.md](📤GitHub上传指南.md)
