## ✨ 核心功能详解

### 1. 📸 拍照识别食物

**功能流程：**
1. 点击"拍照识别食物"进入
2. 点击"开启摄像头"（需要授权）
3. 对准食物拍照
4. AI自动识别（2秒）
5. 显示营养成分
6. 可调整份量和餐次
7. 保存到饮食记录

**技术实现：**
- 使用 `navigator.mediaDevices.getUserMedia()` 调用真实摄像头
- 支持前后摄像头切换
- 支持从相册选择照片
- Canvas截图保存

**注意事项：**
- 首次使用需要授权摄像头权限
- 建议使用HTTPS或localhost
- 部分浏览器可能限制摄像头访问

### 2. 👨‍🍳 AI食谱生成

**功能流程：**
1. 输入想吃的食物（如"炸鸡"）
2. 设置目标卡路里
3. 选择饮食偏好（低脂/高蛋白/素食/无麸质）
4. AI生成替代食谱（2秒）
5. 查看完整食谱：
   - 食材清单
   - 制作步骤
   - 营养对比
   - 健康分析
6. 收藏或重新生成

**AI生成逻辑：**
- 基于用户输入的食物
- 考虑卡路里目标
- 结合饮食偏好
- 生成健康替代方案

### 3. 😊 情绪管理

**功能流程：**
1. 选择当前情绪（8种）
   - 😊 开心
   - 😢 难过
   - 😰 压力
   - 😟 焦虑
   - 😑 无聊
   - 😴 疲惫
   - 😠 生气
   - 😌 平静

2. 记录进食欲望
   - ✅ 没有
   - 😐 轻微
   - 😋 强烈

3. 写简短日记（可选）

4. AI分析并给出：
   - 个性化干预建议
   - 情绪模式分析
   - 触发因素识别

**干预建议示例：**
- 压力：深呼吸、散步、听音乐
- 难过：与朋友聊天、写日记、看电影
- 无聊：阅读、运动、新爱好
