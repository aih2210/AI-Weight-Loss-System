# 📱 手机访问指南

## 方式1：局域网访问（推荐✨）

### 步骤1：启动本地服务器

在项目目录下运行：

```bash
cd E:\项目测试
python -m http.server 8080
```

### 步骤2：获取电脑IP地址

```bash
ipconfig
```

找到 `IPv4 地址`，例如：`192.168.1.100`

### 步骤3：手机访问

确保手机和电脑在同一WiFi网络下，在手机浏览器输入：

```
http://192.168.1.100:8080/mobile-app-premium.html
```

## 方式4：使用ngrok（穿透内网）

### 安装ngrok

1. 下载：https://ngrok.com/download
2. 解压到任意目录

### 启动服务

```bash
# 启动本地服务器
python -m http.server 8080

# 新开一个终端，启动ngrok
ngrok http 8080
```

### 获取公网地址

ngrok会显示一个公网地址，例如：
```
https://xxxx-xx-xx-xx-xx.ngrok.io
```

手机访问：
```
https://xxxx-xx-xx-xx-xx.ngrok.io/mobile-app-premium.html
```

## 🔧 常见问题

**Q: 手机无法访问？**
A: 检查：
1. 手机和电脑是否在同一WiFi
2. 防火墙是否允许8080端口
3. IP地址是否正确

**Q: 页面显示不正常？**
A: 
1. 使用Chrome浏览器
2. 清除浏览器缓存
3. 检查网络连接

**Q: 如何保存到手机？**
A: 
1. 使用PWA版本
2. 浏览器菜单 → 添加到主屏幕
3. 即可像App一样使用
