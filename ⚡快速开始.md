# ⚡ 快速开始 - AI减重助手独立版

## 🚀 3步开始使用

### 第1步：启动服务器

双击运行：`启动本地服务器.bat`

或者在命令行执行：
```bash
python -m http.server 8080
```

### 第2步：获取访问地址

双击运行：`获取IP地址.bat`

或者查看文件：`📱手机访问地址.txt`

你的访问地址是：
```
http://172.20.10.6:8080/standalone-app.html
```

### 第3步：手机访问

1. 确保手机和电脑在同一WiFi
2. 在手机浏览器输入上面的地址
3. 开始使用！

## 📱 推荐：安装为App

### iOS (Safari)
1. 打开链接
2. 点击分享按钮 📤
3. 选择"添加到主屏幕"
4. 完成！

### Android (Chrome)
1. 打开链接
2. 点击菜单 ⋮
3. 选择"安装应用"
4. 完成！

安装后可以完全离线使用！

## 💾 数据说明

### 自动保存
所有操作都会自动保存到本地：
- ✅ 体重记录
- ✅ 饮食记录
- ✅ 运动记录
- ✅ 情绪记录

### 数据备份
1. 进入"个人中心"
2. 点击"数据管理"
3. 点击"导出数据"
4. 保存JSON文件

### 数据恢复
1. 进入"个人中心-数据管理"
2. 点击"导入数据"
3. 选择备份文件
4. 确认导入

## 🎯 使用技巧

1. **每天打卡**
   - 至少记录一次饮食或运动
   - 保持连续打卡

2. **定期称重**
   - 建议每周固定时间
   - 点击首页体重卡片更新

3. **记录情绪**
   - 感到进食欲望时记录
   - 获得AI干预建议

4. **定期备份**
   - 每周导出一次数据
   - 保存到云盘
