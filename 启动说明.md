## 前端启动步骤

由于你的系统没有安装 Node.js，需要先安装：

### 1. 安装 Node.js

访问 https://nodejs.org/ 下载并安装 Node.js LTS 版本（推荐 18.x 或 20.x）

### 2. 安装前端依赖

```bash
cd E:\项目测试\frontend
npm install
```

### 3. 启动前端开发服务器

```bash
npm run dev
```

前端将运行在: http://localhost:3000
